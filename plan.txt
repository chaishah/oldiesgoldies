Project Canvas: Retro-Tech AcademyConcept: A zero-friction, local-first learning app for seniors styled in a 1950s-1960s aesthetic.1. The Tech Stack (Ultra-Lightweight)Frontend: Next.js 15 (App Router).Styling: Tailwind CSS (for that mid-century paper/bakelite look).Data/Progress: localStorage (via Zustand Persist). No Login required.Animations: Framer Motion (for smooth "TV dial" module switching).Hosting: Vercel (Static export).2. Design Identity (1950s-1960s)The Theme: "The Golden Era" of television and radio.Colors: Cream (#F5F5DC), Robin's Egg Blue, and Mustard Yellow.Fonts: Playfair Display (Headlines) and Courier Prime (Typewriter body text).Metaphor: The dashboard is a Vintage Wood-Grain TV. Users turn a "dial" to change lessons.3. The Curriculum (Gamified)ModuleReal-World SkillGamified Metaphor01Touch Gestures (Tap/Swipe)"The Switchboard" - Plugging in phone cables.02Cybersecurity & Scams"The Neighborhood Watch" - Sorting mail from "Snake Oil Salesmen."03Passwords & Privacy"The Master Key" - Locking the front door and "Party Line" safety.4. How "Saving" Works (Technical)Since there is no database or login, the app uses the browser's memory:Storage: User progress is saved as a JSON object in the browser’s localStorage.Logic: Every time a user finishes a module, a "Punch Card" state is updated.Result: High privacy, zero server costs, and zero passwords for the user to forget.5. Implementation RoadmapStep 1: Set up Next.js and the "TV" layout shell.Step 2: Integrate Zustand to remember the user's name and progress.Step 3: Build the "Switchboard" mini-game using SVG drag-and-drop.Step 4: Design "Postal Stamp" rewards for each completed section.Step 5: Deploy to Vercel for free.Quick Tip: To get that authentic 1960s look in Tailwind, use bg-stone-50 for the background and add a slight opacity-90 to your images to make them look like printed paper.Does this roadmap give you enough to start building, or would you like to see a code snippet for the "TV Dial" navigation?
